# 작업 환경 및 규칙

## 로컬 저장소는 휘발성입니다
이 프로젝트는 교육장 공용 Mac에서 작업합니다. **로그아웃하면 로컬에 저장된 모든 파일이 사라집니다** — 이 git 작업 디렉토리 자체도 포함됩니다. `git commit`은 로컬 `.git` 안에만 기록되므로, 그것만으로는 작업이 보존되지 않습니다.

**`git push`로 GitHub(`origin/main`)까지 올라간 것만** 로그아웃 이후에도 남습니다.

## 작업 규칙
- 코드를 수정한 뒤에는, 세션이 끝나기 전(또는 사용자가 로그아웃하기 전)에 **커밋 + 푸시까지 완료할 것을 먼저 제안**한다. 사용자가 잊고 있다가 로그아웃해서 작업을 잃는 일이 없도록 한다.
- "저장됨"과 "안전함"을 동일시하지 않는다 — push되지 않은 변경사항은 이 환경에서 세션이 끝나면 언제든 사라질 수 있다.
- 이후 세션에서 이 저장소를 다시 열게 되면(다른 기기/초기화된 기기에서 `git clone`으로 새로 받는 상황 포함), 위 사실을 다시 상기하고 동일한 규칙을 적용한다.
